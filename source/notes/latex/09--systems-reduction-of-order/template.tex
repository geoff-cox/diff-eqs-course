\noindent  So now we know how to use Euler's Method to solve
 \begin{itemize}
 \item  a single, first-order differential equation (with an initial condition)
 \item {\it systems} of first-order differential equations (with initial conditions).
 \end{itemize}

\noindent  What if we have a higher-order DE, and we can't use any of the solution techniques
we have studied so far?  Is there some way we can leverage this numerical technique to get a
solution?  Turns out there is!  We just need to use a simple change of variables called
{\bf reduction of order.}\\ \vspace{1cm}

\noindent  {\bf Summary:}
\begin{itemize}
\item  Analytic solutions are exact solutions, and having {\it the} exact solution is usually
 the gold standard we aim for...
\item ...but...we only have a finite number of techniques to solve differential equations, and there
 are far more differential equations for which we have no analytic solution technique.
\item  In many instances, we are content with a {\bf numerical solution,} where we approximate the
 value of the solution.  Numerical solutions can be generated quickly and can solve DEs
 and systems of DEs when we can't find analytic solutions.
\item  {\bf Euler's Method} is an example of one numerical solution technique.  It gives a crude
 approximation based on the idea of tangent lines
\item  Euler's Method works for first-order DEs and systems of first-order DEs.  If we have a higher
 order DE or system, we can make a change of variables we call {\bf reduction of order} so
 that we can use Euler's Method.
\item  MANY of the other commonly used numerical solution algorithms (including those used
 in the solvers of the best software available) are variations on Euler's Method.
\end{itemize}
